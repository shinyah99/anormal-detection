# anormal-detection
## Overview
Anormal Detection for Web Server Access Logs using Machine Learning. Several anormal detection model was built.

* Isolation Forest
* kNN (using pyod)
* INNE (using pyod)
* LODA (using pyod)
* ABOD (using pyod)
* AutoEncoder (using pyod)
* DeepSVDD (using pyod)
* DBSCAN

## Dataset
